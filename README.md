# Deepfake Detector

A multimodal deepfake detection application for videos, images, and audio modalities. 
Powered by Gradio.

## Features
- **Image Inference**: Detects generated/fake images.
- **Video Inference**: Analyzes videos for deepfakes.
- **Audio Inference**: Detects AI-generated or altered audio.

## Installation & Running Locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python app.py
   ```

The application will start on a local server (typically `http://127.0.0.1:7860/`).
